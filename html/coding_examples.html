Coding Examples

The plugin Typed.js was used to create the typewriter effect. You will find an example of this code below.

const typed = new Typed(".auto-type", { strings: ["I am a Web Developer", "I Enjoy Learning", "I Enjoy Problem Solving", "I am a Web Developer"], startDelay: 1000, typeSpeed: 150, backSpeed: 100, backDelay: 1000, smartBack: true });
